## 9. 최적화 체크리스트

### 기술 SEO
- [ ] 모든 페이지에 canonical URL 설정
- [ ] XML 사이트맵 생성 및 Google Search Console 등록
- [ ] robots.txt 설정 (Disallow: /admin, /private)
- [ ] 페이지 로딩 속도 < 3초 (모바일 < 4초)
- [ ] 모든 이미지 WebP 최적화 (og:image 제외, jpg 사용)
- [ ] 모든 링크 내부·외부 구분 및 nofollow 설정 검토

### 온페이지 SEO
- [ ] H1 태그 1개만 사용 (title과 유사)
- [ ] H2-H3 계층 구조 올바르게 구성
- [ ] 키워드 밀도 1-2% 유지
- [ ] LSI 키워드 자연스럽게 포함
- [ ] 이미지 ALT 텍스트 모두 작성 (50자 이내)
- [ ] 내부 링크 최소 3-5개 (관련 페이지)
- [ ] 외부 참고 링크 1-2개 (신뢰도 향상)
