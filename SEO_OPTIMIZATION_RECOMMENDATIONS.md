## VERIFICATION CODES - SETUP GUIDE

### Naver Search Advisor

1. Visit: https://searchadvisor.naver.com/
2. Login with account
3. Click "Domains" or "사이트 등록"
4. Add each region domain:
   - high-karaoke.com
   - hikaraoke.com
   - best-karaoke.com
   - etc.
5. Copy verification code format:
   ```
   <meta name="naver-site-verification" content="naver****32character****code">
   ```
6. Extract code and update region.ts:
   ```typescript
   naverVerification: 'naverfb8a1234....' // 32-char code
   ```

### Google Search Console

1. Visit: https://search.google.com/search-console/
2. Add property: Select "URL prefix" option
3. Enter: https://{region-domain}
4. Use "HTML tag" verification method
5. Copy code format:
   ```
   <meta name="google-site-verification" content="google1234...code">
   ```
6. Extract code and update region.ts:
   ```typescript
   googleVerification: 'google1234....' // 43-char code
   ```
