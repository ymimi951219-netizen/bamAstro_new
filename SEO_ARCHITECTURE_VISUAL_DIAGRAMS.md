# SEO 아키텍처 시각화 & 구현 로드맵

## 1. 사이트 구조 맵 (Current vs Proposed)

### 현재 구조 (Linear)

```
┌─────────────────────────────────────────────────────────────────┐
│                        HOMEPAGE [/]                             │
│         "분당 유흥 정보 가이드" (Generic Hub)                    │
└─────────────────────────────────────────────────────────────────┘
                    ↓ ↓ ↓ ↓ ↓ ↓
    ┌───────────────────────────────────────┬──────────────────────────────────┐
    │      PILLAR PAGES (6개 업종)           │      SECONDARY (2개 비교 + 가격)   │
    ├───────────────────────────────────────┼──────────────────────────────────┤
    │ • 가라오케 가이드                      │ • 가라오케 vs 하이퍼블릭             │
    │ • 하이퍼블릭 가이드                    │ • 가격 가이드                      │
    │ • 호스트바 가이드                      │                                   │
    │ • 룸살롱 가이드                        │                                   │
    │ • 셔츠룸 가이드                        │                                   │
    │ • 기모노룸 가이드                      │                                   │
    └───────────────────────────────────────┴──────────────────────────────────┘
                    ↓ (각 페이지에서 /faq로만 링크)
    ┌──────────────────────────────────────────────────────────────┐
    │             FAQ 페이지 (6개)                                  │
    │          각 업종별 FAQ (3-5개 질문만)                         │
    └──────────────────────────────────────────────────────────────┘
                    ↓ (단방향)
    ┌──────────────────────────────────────────────────────────────┐
    │                BLOG (동적)                                    │
    │            월 1-2개 포스트 (느슨함)                           │
    └──────────────────────────────────────────────────────────────┘

⚠️ 문제점:
- 단일 지역 중심 (다지역 미활용)
- 선형 구조: 지역→업종→가격→블로그 (순환 링크 없음)
- 주변역별 페이지 부재
- 상황별 가이드 부재 (회식/데이트/초보자 구분 X)
- 비교 페이지 극소수 (2개만)
- FAQ 깊이 부족 (3-5개)
- 대부분의 페이지가 고아 상태
```
