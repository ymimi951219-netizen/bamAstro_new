```
□ Page Speed Insights: 90점 이상
□ Core Web Vitals: 모두 "Good"
□ 이미지 최적화: WebP 포맷, lazy loading

모바일:
□ 모바일 반응형 테스트 통과
□ 모바일 사용성: 클릭 요소 충분한 크기
□ 뷰포트 메타 태그: 설정됨
```

### D.2 콘텐츠 SEO 체크리스트

```
키워드:
□ 주요 키워드: 도입+H1+본론에 2-3회
□ LSI 키워드: 자연스럽게 3-5개 포함
□ 롱테일 키워드: 특정 검색의도 대상 3-4개
□ 키워드 밀도: 1-2% (과도하지 않음)

내용:
□ Word Count: 목표 길이 달성 (±10%)
□ 읽기 쉬움: 평균 문장 길이 25단어 이하
□ 이미지: 200자 이상의 콘텐츠마다 1개
□ 비디오/임베드: 1-2개 포함 (가능시)

구조:
□ 명확한 계층: H1 > H2 > H3 (일관성)
□ 각 섹션: 100자 이상
□ 목차 (TOC): 5개 이상 섹션시 필수
□ 내부링크: 5-8개, 자연스럽고 관련성 있음

신뢰성:

□ 사실성: 모든 통계/수치 출처 명확
□ 업데이트 표시: dateModified 스키마 포함
□ 신선한 정보: 최근 데이터/예시 포함
```

## I. 트러블슈팅 가이드

### 문제: 콘텐츠가 색인되지 않음

**진단**:
1. Google Search Console에서 URL 검증
2. robots.txt 확인
3. noindex 메타 태그 확인

**해결책**:
```
1. robots.txt에 페이지가 차단되어 있다면 제거
2. noindex가 설정되어 있다면 제거
3. GSC에서 "요청" → "색인 생성 요청" 수행
4. 24-48시간 대기
```

### 문제: 콘텐츠가 색인되었지만 순위가 없음

**원인 분석**:
- 키워드 관련성 낮음
- 콘텐츠 길이 부족
- 내부링크 부족
- 경쟁 페이지 강함

**개선책**:
```
1. 콘텐츠 강화:
   - 최소 2000자 이상으로 확장
   - LSI 키워드 추가
   - FAQ 섹션 추가

2. 내부링크 추가:
   - 관련 Pillar 페이지에서 링크
   - 5-8개의 자연스러운 링크

3. 외부 신호:
   - 소셜 미디어 공유
   - 커뮤니티 언급

4. 콘텐츠 신선화:
   - dateModified 업데이트
   - 최신 정보 추가
```

### 문제: 이탈율이 높음

**진단**:
1. 페이지 로딩 속도 확인
2. 모바일 반응형 확인
3. 사용자 행동 분석 (스크롤 깊이)

**해결책**:
```
1. 속도 개선:
   - 이미지 압축
   - 캐시 활성화
   - CDN 활용

2. UX 개선:
   - 첫 섹션 강화 (처음 3초 중요)
   - CTA 명확화
   - 내부링크 눈에 띄게 배치

3. 콘텐츠 개선:
   - 서론 길이 줄이기 (150자 이내)
   - 시각자료 추가
   - 리스트/테이블 활용
```
