## 🔑 주요 키워드 통계

### 기본 키워드 범위

```
주 키워드: 72개 (12지역 × 6장르)
예시: "강남 가라오케", "분당 호빠", "수원 룸살롱" 등

보조 키워드: 200-300개
예시: "강남역 가라오케", "강남 가라오케 가이드", "강남 가라오케 가격" 등

LSI 키워드: 30개 (의미적 관련)
예시: 초이스, 풀타임, 주대, TC, 비즈니스 접대 등

블로그 롱테일: 500+개 (5-7단어)
예시: "강남 가라오케 첫 방문 완벽 가이드" 등
```

### 검색 의도별 분포

```
정보형 검색: 60% (초보자 가이드, 비교, 팁)
→ 현재: 우수 (가이드 페이지)
→ 개선: 블로그로 심화 콘텐츠 추가

네비형 검색: 25% (지역/업소 찾기)
→ 현재: 기본 (홈페이지, 가이드 페이지)
→ 개선: 역명 페이지 추가 (지역 세분화)

거래형 검색: 15% (예약)
→ 현재: 우수 (모든 페이지에 CTA)
→ 개선: 가격 정보 섹션 강화
```

## 🎓 한국 시장 특수성

### Naver vs Google 전략
```
Google (60% 검색 점유율):
- Schema.org 마크업 중요
- 모바일 우선 인덱싱
- 콘텐츠 품질 중심

Naver (40% 검색 점유율):
- 정확한 H태그 계층 중요
- 한글 메타 데이터 필수
- 블로그 백링크 효과 높음
- 콘텐츠 업데이트 빈도 중요

권장 전략: Google 중심 최적화 + Naver 메타 추가
```

### 지역 검색 특성
```
한국 사용자는 지역명 + 장르 검색 선호:
✓ "[지역] 가라오케" (자주)
✗ "가라오케 강남" (드물게)
✗ "best karaoke in [region]" (거의 없음)

역명 검색도 상당한 트래픽:
✓ "[역명] 호빠" (100-200/월)
✓ "[역명] 가라오케" (80-150/월)

따라서: 지역+장르+역명 조합 모두 필요
```

## 💬 자주 묻는 질문

### Q1: "지금 바로 블로그를 시작해야 하나?"
**A:** 아니오. 먼저 역명 페이지 기초를 잡은 후 월 5개 포스트로 시작하세요.
최우선은 Week 1-2의 기초 설정과 Month 1-2의 역명 페이지입니다.

### Q2: "Naver SEO가 정말 중요한가?"
**A:** 네. 한국 검색의 40%를 차지합니다. 기본 설정만 5시간이면 되므로 필수입니다.

### Q3: "키워드 밀도를 정확히 몇 %로 해야 하나?"
**A:** 주 키워드 0.8-1.2% (1,000단어당 8-12회). 하지만 가독성이 최우선입니다.
도구로 측정하지 말고, 자연스럽게 읽히는지 확인하세요.

### Q4: "지금 바로 50개 역명 페이지를 만들어야 하나?"
**A:** 아니오. 강남 3개로 시작해서 프로토타입을 검증한 후 확대하세요.

### Q5: "Google AdSense나 광고로 수익화할 수 있나?"
**A:** 가능하지만, 현재 구조상 예약 중심이 더 적합합니다.
Traffic이 월 5,000을 넘으면 AdSense 추가를 검토하세요.
